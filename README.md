# 🪷 Krishna Journal — Static Marketing Website

A complete static marketing website for **Krishna Journal** — a Flutter iOS + Android app that helps users reflect on their emotions through Bhagavad Gita shlokas and guidance from Lord Krishna.

## 🚀 How to Run Locally

No build tools, no npm, no dependencies. Just open in your browser:

```bash
# Option 1 — Simply double-click index.html in your file manager
# Option 2 — Serve with Python's built-in server:
python3 -m http.server 8080
# Then open http://localhost:8080 in your browser
```

---

## 🌐 How to Deploy to GitHub Pages

1. Push all files to the `main` branch of your repository
2. Go to your repository on GitHub
3. Navigate to **Settings → Pages**
4. Under **Source**, select **Deploy from a branch**
5. Choose **Branch: main** and **Folder: / (root)**
6. Click **Save**
7. Your site will be live at `https://chitraguptpandey.github.io/KrishnaJournalWebsite/`
